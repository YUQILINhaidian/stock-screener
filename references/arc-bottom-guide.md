# 圆弧底选股策略详解

## 📌 策略概述

**策略名称**：圆弧底选股  
**策略类型**：底部形态识别  
**适用周期**：日线  
**持仓周期**：中长期（2-8周）  
**适用场景**：底部建仓、趋势反转初期  

---

## 🎯 核心思想

通过构建**四档多周期加权综合均线**，识别股票从底部区域开始反转的特定形态：

```
四档综合线结构
─────────────────────────────────────────────
条件2（短线）  ← EMA(4) + MA(8) + MA(16)   → 最灵敏
条件5（中短）  ← EMA(9) + MA(18) + MA(36)
条件4（中线）  ← EMA(13) + MA(26) + MA(52)
条件6（长线）  ← EMA(24) + MA(48) + MA(96) → 最稳定
─────────────────────────────────────────────

圆弧底 = 短线在底部蛰伏30天后首次上穿中线
        + 长中线收敛（底部形态稳固）
        + 短中期线仍低于长线（仍处于底部区域）
```

---

## 📐 通达信原始公式

```
AA:=(C*2+H+L+O)/5;
条件1:=MA(C,5);
条件2:=(EMA(AA,4)+MA(AA,8)+MA(AA,16))/3;
条件5:=(EMA(AA,9)+MA(AA,18)+MA(AA,36))/3;
条件4:=(EMA(AA,13)+MA(AA,26)+MA(AA,52))/3;
条件6:=(EMA(AA,24)+MA(AA,48)+MA(AA,96))/3;
条件10:=MAX(MAX(条件2,条件5),MAX(条件4,条件6));
条件9:=MIN(MIN(条件2,条件5),MIN(条件4,条件6));
条件16:=MIN(MIN(条件2,条件5),条件4);
条件20:=MAX(MAX(条件2,条件5),条件4);
圆弧底:LONGCROSS(条件2,条件4,30)
       AND (条件6-条件4)/条件4<0.05
       AND 条件20<条件6;
```

---

## 🔬 三个触发条件逐条解析

### 条件一：`LONGCROSS(条件2, 条件4, 30)` — 底部蛰伏后上穿

**含义**：短周期综合线（条件2）在过去30天持续低于中周期综合线（条件4），今天首次上穿。

| 状态 | 含义 |
|------|------|
| 过去30天：条件2 < 条件4 | 股票处于底部调整期，短线一直压在中线下方 |
| 今天：条件2 > 条件4 | 短线首次突破中线，底部反转开始 |

**为什么需要30天？**  
- 太短（如5天）：可能是噪音，不构成圆弧底形态  
- 30天是一个月，意味着股票经历了完整的底部蓄能期

---

### 条件二：`(条件6-条件4)/条件4 < 0.05` — 长中线收敛

**含义**：长周期综合线（条件6）与中周期综合线（条件4）的差距小于5%。

```
收敛率 = (长线 - 中线) / 中线 × 100% < 5%
```

**意义**：
- 表示多个中长期均线已经靠拢并平行运行
- 说明股票在底部区域横盘整理，价格形成稳固底部
- 收敛越小，底部形态越扎实

---

### 条件三：`条件20 < 条件6` — 短中期仍在长线下方

**含义**：短中期最大值（MAX(条件2, 条件5, 条件4)）仍低于长周期综合线（条件6）。

**意义**：
- 股票虽然短线上穿了中线，但整体仍处于底部区域
- 长期均线尚在短中期线上方，有上升空间
- 是"刚开始反转"而非"已经大幅上涨"的保证

---

## 📊 指标含义速查

| 指标 | 计算 | 含义 |
|------|------|------|
| `AA` | `(C×2 + H + L + O) / 5` | 加权均价（收盘价权重更高） |
| `条件2` | `(EMA(AA,4) + MA(AA,8) + MA(AA,16)) / 3` | 短周期三均线综合（4/8/16日） |
| `条件5` | `(EMA(AA,9) + MA(AA,18) + MA(AA,36)) / 3` | 中短周期三均线综合（9/18/36日） |
| `条件4` | `(EMA(AA,13) + MA(AA,26) + MA(AA,52)) / 3` | 中周期三均线综合（13/26/52日） |
| `条件6` | `(EMA(AA,24) + MA(AA,48) + MA(AA,96)) / 3` | 长周期三均线综合（24/48/96日） |
| `条件20` | `MAX(条件2, 条件5, 条件4)` | 短中期三线的最大值 |

## 💡 使用建议

### ✅ 适合的买入时机
1. 信号触发后次日低开观察，确认缩量
2. 配合K线图看是否有实体大阳线
3. 优先选择收敛率（converge_ratio）更小的股票

### ⚠️ 注意事项
1. **圆弧底是买点预警，不是立即买入信号**
2. 建议配合大盘方向判断（牛市效果更好）
3. 设置止损位（-5%到-8%）
4. 信号稀少时，可适当放宽收敛阈值至8%

### 🔗 组合策略
- **配合RPS筛选**：优先选择 RPS120 > 80 的标的
- **配合火车头策略**：两者均命中的股票更可靠
- **配合口袋支点**：等待圆弧底后出现口袋支点买入
