# 火车头策略详细指南

## 策略概述

火车头策略是陶博士2006验证的有效策略，包含两个子策略：MRGC（美人国）和SXHCG（双线红）。

## MRGC策略（美人国）

### 基础条件（必须满足）
- 换手率 < 25%
- 120天最大回撤 ≤ 50%
- 收盘价 > 250天最高价的70%

### 子策略详解

#### XG1: 创新高 + 高RPS（最激进）
- 股价创新高
- RPS120 ≥ 95 或 RPS250 ≥ 95
- 适合：追求强势突破的投资者
- 风险：较高，可能在高位买入

#### XG2: 强势位置 + 超高RPS
- 股价处于强势位置
- RPS120 ≥ 97 且 RPS250 ≥ 97
- 适合：追求确定性的投资者
- 风险：中等

#### XG3: 次强势 + 极高RPS（推荐）
- 股价次强势
- RPS120 ≥ 98 或 RPS250 ≥ 98
- 适合：大多数投资者
- 风险：中等偏低

#### XG4: 低回撤 + 高位 + 高RPS（最稳健）
- 120天最大回撤较小
- 股价处于相对高位
- RPS120 ≥ 95 且 RPS250 ≥ 95
- 适合：稳健型投资者
- 风险：最低

## SXHCG策略（双线红）

### 筛选条件（全部满足）
1. RPS120 + RPS250 > 185
2. 均线多头排列（MA10 > MA20 > MA50 > MA120）
3. 20天最大回撤 ≤ 25%
4. 10日线、20日线向上
5. 换手率 < 15%
6. 120天最大回撤 ≤ 50%

### 适用场景
- 波段交易
- 追求短期收益
- 捕捉趋势延续

## 策略组合使用

### 最佳实践
1. 首先运行MRGC筛选
2. 然后运行SXHCG筛选
3. 重点关注同时符合两个策略的股票
4. 根据个人风格选择子策略

### 历史表现
- 2026-02-26测试：538只股票
- 2026-03-04测试：41只股票（市场走弱）
- 平均RPS和：188.8
- 建议配置：Top 5-10只

## 风险提示

1. 市场环境变化会影响筛选结果数量
2. 创新高股票风险较高，注意止损
3. 建议分散投资，不要集中持仓
4. 关注市场整体趋势，顺势而为
