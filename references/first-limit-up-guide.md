# 首次涨停&一线红选股策略详解

## 📌 策略概述

**策略名称**：首次涨停&一线红  
**策略类型**：涨停突破 + RPS强度筛选  
**适用周期**：日线  
**持仓周期**：短线（1-5天）  
**适用场景**：短线追涨、捕捉涨停突破  

---

## 🎯 核心思想

结合**RPS相对强度**和**涨停板信号**，捕捉强势股的首次涨停突破机会：

```
RPS强度筛选 → 强势股池
     ↓
涨停板触发 → 突破信号
     ↓
一线红确认 → 价格位置验证
     ↓
首次触发 → 捕捉启动时机
```

---

## 📐 通达信原始公式

```
{RPS要素}
N:=90;
RPS50:=EXTDATA_USER(3,0)/10>=N;   {RPS50大于等于90}
RPS120:=EXTDATA_USER(1,0)/10>=N;  {RPS120大于等于90}
RPS250:=EXTDATA_USER(2,0)/10>=N;  {RPS250大于等于90}

{涨停}
ZDF:=IF(INBLOCK('创业板'), 0.2, IF(INBLOCK('科创板'),0.2,IF(INBLOCK('ST板块'), 0.05, 0.1)));
ZT:=ZTPRICE(REF(C,1),ZDF);
涨停:=C=ZT AND L<H;

XG:=(RPS50 OR RPS120 OR RPS250) AND H/HHV(HIGH,250)>0.5 AND 涨停;
BARSSINCEN(XG,20)=0;
```

---

## 🔬 四个条件逐条解析

### 条件一：RPS强度（任一 ≥ 90）

**含义**：RPS50、RPS120、RPS250 三者中至少有一个 ≥ 90

| RPS指标 | 含义 |
|---------|------|
| RPS50 ≥ 90 | 50日相对强度超过90%的股票 |
| RPS120 ≥ 90 | 120日相对强度超过90%的股票 |
| RPS250 ≥ 90 | 250日相对强度超过90%的股票 |

**意义**：
- 确保股票具有相对强度优势
- 过滤弱势股的涨停（可能是反弹而非反转）
- 中长期强势股的涨停更有持续性

---

### 条件二：涨停板

**含义**：当日收盘价等于涨停价，且有实际交易

**涨停幅度**：

| 板块 | 涨停幅度 |
|------|----------|
| 创业板（300xxx） | 20% |
| 科创板（688xxx） | 20% |
| ST板块 | 5% |
| 主板（其他） | 10% |

**涨停价计算**：
```
涨停价 = 昨日收盘价 × (1 + 涨停幅度)
```

**交易条件**：`L < H`（有实际换手，非一字涨停）

---

### 条件三：一线红

**含义**：最高价 ≥ 250日最高价 × 50%

```
一线红条件 = 当日最高价 ≥ 250日最高价 × 0.5
```

**意义**：
- 确保股价处于相对高位区域
- 过滤低位涨停（可能是超跌反弹）
- 高位涨停意味着突破或加速

---

### 条件四：首次触发

**含义**：过去20天内首次满足上述条件

```
BARSSINCEN(XG, 20) = 0
```

**意义**：
- 捕捉首次涨停突破
- 避免追高连续涨停
- 首次涨停通常有更高的成功率

## ⚠️ 使用建议

### ✅ 适合的情况
1. 大盘处于上升趋势或震荡市
2. 筛选结果数量适中（20-50只）
3. 次日开盘有溢价机会

### ❌ 需要谨慎的情况
1. 大盘处于下跌趋势
2. 筛选结果过多（>80只）可能意味着市场过热
3. 涨停后次日低开

### 📌 操作要点
1. **次日观察**：不要盲目追高，观察开盘情况
2. **仓位控制**：单只股票 ≤ 10% 仓位
3. **止损纪律**：严格执行 -5% 止损
4. **分批建仓**：首次建仓 50%，确认趋势后加仓
