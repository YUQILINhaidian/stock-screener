# 老鸭头形态识别策略

## 一、策略概述

老鸭头形态是唐能通先生在《短线是银》系列中提出的经典K线形态,是捕捉主升浪的重要信号。该形态完整描述了主力建仓、洗盘、拉升的全过程。

### 形态特征

```
价格
 ^
 |    ③鸭头
 |     /\      ⑤鸭嘴张开(买入点)
 |    /  \    /
 |   /    \  /
 |  /      \/
 | /      ④鸭颈(洗盘)
 |/________________> 时间
   ①建仓期
```

---

## 二、形态识别规则

### 阶段1:建仓期(60日均线向上)

**特征**:
- 股价沿5日、10日、60日均线缓慢上升
- 均线呈多头排列:MA5 > MA10 > MA60
- 60日均线斜率向上,表明中期趋势良好

**意义**: 主力在此阶段悄悄吸筹,股价稳步上行

---

### 阶段2:鸭头形成(30-40天前)

**特征**:
- 股价快速拉升至阶段性高点
- 高点位置通常在30-40天前
- 这个高点就是"鸭头"

**意义**: 主力建仓完成后试探性拉升

**识别条件**:
```python
# 找到60天内最高点
high_60 = max(high[-60:])

# 鸭头高点距今30-40天
30 <= days_from_high <= 40
```

---

### 阶段3:鸭颈形成(回调洗盘)

**特征**:
- 股价从鸭头回调
- **跌破5日线和10日线**(重要!)
- **但不深度跌破60日均线**(允许5%偏差)
- 回调幅度:8%-30%

**意义**: 主力洗盘,清洗获利盘和不坚定筹码

**识别条件**:
```python
# 1. 跌破短期均线
close[低点] < ma5[低点]
close[低点] < ma10[低点]

# 2. 守住60日均线
close[低点] > ma60[低点] * 0.95

# 3. 回调幅度合理
drop_rate = (鸭头价格 - 鸭颈价格) / 鸭头价格
8% <= drop_rate <= 30%
```

---

### 阶段4:鸭嘴张开(买入信号)

**特征**:
- 股价重新向上突破前期高点(鸭头)
- 股价站上5日线和10日线
- 成交量明显放大(≥1.1倍10日均量)
- 均线重新多头排列

**意义**: 洗盘结束,主升浪开始!

**识别条件**:
```python
# 1. 接近或突破前高
close[-1] > 鸭头价格 * 0.95

# 2. 站上短期均线
close[-1] > ma5[-1]
close[-1] > ma10[-1]

# 3. 成交量放大
volume[-1] > volume_ma10[-1] * 1.1

# 4. 均线多头排列
ma5[-1] > ma10[-1] > ma60[-1]
```

## 三、策略参数

| 参数 | 默认值 | 说明 | 调整范围 |
|------|--------|------|----------|
| `DUCK_HEAD_DAYS_MIN` | 20 | 鸭头最小距离(天) | 15-30 |
| `DUCK_HEAD_DAYS_MAX` | 50 | 鸭头最大距离(天) | 40-60 |
| `DROP_RATE_MIN` | 8% | 最小回调幅度 | 5-10% |
| `DROP_RATE_MAX` | 30% | 最大回调幅度 | 25-35% |
| `MA60_DEVIATION` | 0.95 | 60日线允许偏差 | 0.93-0.98 |
| `VOLUME_RATIO_MIN` | 1.1 | 成交量放大倍数 | 1.0-1.3 |
| `BREAKOUT_THRESHOLD` | 0.95 | 突破前高阈值 | 0.92-0.98 |

## 五、交易策略

### 买入策略

**最佳买点**: 鸭嘴张开时(突破前高+成交量放大)

**买入条件**:
1. ✅ 股价突破前期高点(鸭头)
2. ✅ 成交量明显放大(≥1.1倍)
3. ✅ 均线多头排列
4. ✅ 大盘环境良好(最好是多头市场)

**分批建仓**:
- 第一笔:突破日买入50%仓位
- 第二笔:确认有效突破后加仓30%
- 第三笔:留20%机动仓位

---

### 止损策略

| 止损位置 | 止损幅度 | 说明 |
|----------|----------|------|
| **鸭颈位下方** | -5% | 标准止损 |
| **60日均线下方** | -8% | 保守止损 |
| **破位止损** | -10% | 最大容忍 |

**重要**: 止损位设置后必须严格执行,不能心存侥幸!

---

### 止盈策略

| 涨幅 | 操作 | 说明 |
|------|------|------|
| +15% | 卖出30% | 锁定部分利润 |
| +30% | 卖出30% | 继续持有核心仓位 |
| +50% | 卖出30% | 让利润奔跑 |
| 跌破10日线 | 清仓 | 趋势反转信号 |

## 七、常见问题

### Q1: 如何区分假突破?

**A**: 看成交量!
- ✅ 真突破:成交量放大≥1.5倍,且连续3天站稳
- ❌ 假突破:缩量突破,或突破后快速回落

---

### Q2: 回调跌破60日线怎么办?

**A**: 分情况处理:
- **轻微跌破**(< 5%): 可以容忍,关注能否快速收回
- **深度跌破**(> 8%): 形态破坏,不再是老鸭头

---

### Q3: 鸭嘴张开但没买到怎么办?

**A**: 
- 不要追高!等待回踩5日线或10日线
- 如果回踩不破均线,可以考虑补仓
- 如果连续大涨,放弃这只股票,等待下一个机会

---

### Q4: 弱市中老鸭头失效率高吗?

**A**: 是的!
- **牛市**:成功率70-80%
- **震荡市**:成功率50-60%
- **熊市**:成功率< 40%,不建议使用

**建议**: 只在大盘趋势良好时使用该策略

## 八、风险提示

### ⚠️ 形态陷阱

| 陷阱类型 | 特征 | 应对 |
|----------|------|------|
| **假鸭头** | 前期没有明显建仓期 | 放弃 |
| **深V反转** | 鸭颈跌破60日线过多 | 不是标准老鸭头 |
| **缩量突破** | 鸭嘴张开时量能不足 | 等待放量确认 |
| **连续涨停** | 无法买入 | 放弃,不追高 |

---

### ⚠️ 市场环境风险

| 市场状态 | 建议 |
|----------|------|
| **牛市** | ✅ 积极参与 |
| **震荡市** | ⚠️ 谨慎操作,重点关注龙头股 |
| **熊市** | ❌ 暂停使用,保持空仓 |

---

### ⚠️ 仓位控制

- **单只股票**:≤ 10%总资金
- **老鸭头策略总仓位**:≤ 30%
- **分散持股**:3-5只为宜,避免过度分散

- 查看最近1个月行业涨幅排行

- 市值 > 50亿(流动性好)

## 十一、总结

老鸭头形态是一个**完整的主力操作轨迹**:

```
建仓 → 拉升 → 洗盘 → 再拉升(主升浪)
```

**关键点**:
1. ✅ 鸭颈不能破60日线(洗盘不能太深)
2. ✅ 鸭嘴张开必须放量(主力真实拉升)
3. ✅ 大盘环境必须良好(顺势而为)
4. ✅ 严格止损(形态破坏立即退出)

**适用人群**:
- 中短期投资者
- 能够及时盯盘的交易者
- 有一定技术分析基础的投资者

**预期收益**: 单只股票15-30%(1-2个月)
