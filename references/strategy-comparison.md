## 🔍 详细逻辑对比

### 1. 火车头策略（MRGC + SXHCG）

#### 核心逻辑
**捕捉持续强势股，在趋势确认时介入**

#### 筛选条件分解

**MRGC（美人国）- 中长期策略**：
```
基础门槛：
├─ 换手率 < 25%（避免过度炒作）
├─ 120天最大回撤 ≤ 50%（风险控制）
└─ 收盘价 > 250天最高价的70%（保持强势）

子策略（满足任一）：
├─ XG1: 创新高 + RPS120/250≥95
├─ XG2: 强势位置 + RPS≥97
├─ XG3: 次强势 + RPS≥98 ⭐推荐
└─ XG4: 低回撤 + 高位 + RPS≥95 ⭐稳健
```

**SXHCG（双线红）- 波段策略**：
```
必须全部满足：
├─ RPS120 + RPS250 > 185（双线强势）
├─ 均线多头排列（趋势向上）
├─ 20天最大回撤 ≤ 25%（短期风险控制）
├─ 10/20日线向上（动能确认）
├─ 换手率 < 15%（避免追高）
└─ 120天最大回撤 ≤ 50%（中长期风险）
```

#### 优势
- ✅ 双重策略互补（中长期+波段）
- ✅ 已验证有效（胜率78%，平均收益+4.33%）
- ✅ 多个子策略可选
- ✅ 包含风险控制

#### 劣势
- ⚠️ 条件较复杂
- ⚠️ 需要换手率数据
- ⚠️ 在弱势市场筛选结果减少

---

### 2. 三线红策略

#### 核心逻辑
**最简单的强势股筛选，RPS三线共振**

#### 筛选条件分解
```
必须全部满足：
├─ RPS5 > 90（短期强势）
├─ RPS120 > 90（中期强势）
└─ RPS250 > 90（长期强势）
```

#### 优势
- ✅ 最简单直接
- ✅ 纯技术筛选，无需其他数据
- ✅ 快速执行
- ✅ 容易理解

#### 劣势
- ⚠️ 条件单一，无风险控制
- ⚠️ 不考虑买入时机
- ⚠️ 可能错过调整后的机会

---

### 3. 蓝色钻石策略

#### 核心逻辑
**捕捉强势股调整后的第二波上涨机会**

#### 筛选条件分解
```
ZCDX00 - 基础条件：
├─ 20天最大回撤 ≤ 25%
└─ 收盘价 > 250天最高价的80%

ZCDX01 - RPS条件（满足任一）：
├─ RPS50 ≥ 98
├─ RPS20 ≥ 98
└─ RPS50≥97 且 RPS20+RPS50≥190

ZCDX02 - 靠近20日线：
└─ 收盘价/MA20 < 1.005

ZCDX03 - 均线位置健康：
├─ 过去20天低于20日线≤2天
└─ 过去20天低于10日线≤8天

ZCDX04 - MA50领先：
└─ MA50 > MA120/200/250

ZCDX05 - 换手率控制：
└─ 换手率 < 10%
```

#### 优势
- ✅ 不追高，寻找调整机会
- ✅ 使用短期RPS（20/50），反应更快
- ✅ 风险相对较低
- ✅ 明确的买入信号（靠近MA20）

#### 劣势
- ⚠️ 需要完整的换手率数据
- ⚠️ 条件较严格，筛选结果可能较少
- ⚠️ 未经过充分验证

---

### 4. RPS + 基本面组合

#### 核心逻辑
**技术面强势 + 基本面低估，双重验证**

#### 筛选条件分解
```
技术面：
├─ RPS120 > 阈值（可调，如90）
└─ RPS250 > 阈值（可调，如90）

基本面（可选）：
├─ PE市盈率 < 阈值
├─ PB市净率 < 阈值
├─ ROE > 阈值
└─ 营收增长率 > 阈值
```

#### 优势
- ✅ 技术+基本面双重安全边际
- ✅ 参数可自定义
- ✅ 适合价值投资
- ✅ 降低纯技术风险

#### 劣势
- ⚠️ 需要基本面数据支持
- ⚠️ 可能错过纯技术机会
- ⚠️ 数据更新频率较低

## 🎯 核心差异对比

### RPS使用差异

| 策略 | 主要RPS | 辅助RPS | 说明 |
|------|---------|---------|------|
| **火车头** | 120, 250 | 5, 50 | 中长期强势确认 |
| **三线红** | 5, 120, 250 | - | 三线共振，全面强势 |
| **蓝色钻石** | 20, 50 | - | 短中期强势，反应快 |
| **RPS+基本面** | 120, 250 | - | 中长期强势 |

**关键差异**：
- 火车头、三线红、RPS+基本面使用 **中长期RPS**（120/250）
- 蓝色钻石使用 **短期RPS**（20/50），更适合捕捉短期机会

---

### 买入时机差异

| 策略 | 买入时机 | 特点 |
|------|----------|------|
| **火车头** | 创新高或强势位置 | 追强势，适合趋势延续 |
| **三线红** | 不考虑时机 | 纯筛选，需自己判断 |
| **蓝色钻石** | 回调至MA20附近 ⭐ | 不追高，等回调 |
| **RPS+基本面** | 不考虑时机 | 需结合其他指标 |

**关键差异**：
- 蓝色钻石有**明确的买入信号**（靠近MA20）
- 其他策略主要筛选股票，买入时机需自行判断

## ⚠️ 注意事项

1. **策略组合使用**
   - 不要只用单一策略
   - 多策略交叉验证效果更好
   - 同时符合多个策略的股票更可靠

2. **市场环境判断**
   - 牛市可以用激进策略
   - 震荡市用蓝色钻石
   - 熊市减少仓位或用RPS+基本面

3. **风险控制**
   - 单只股票仓位≤10%
   - 总仓位根据市场调整
   - 严格执行止损（-3%到-5%）

4. **数据质量**
   - 确保数据是最新的
   - 定期更新K线和RPS数据
   - 蓝色钻石需要完整的换手率数据
