# 第二阶段策略使用示例

## 示例 1：基础筛选

```bash
# 执行第二阶段筛选
python3 ~/screen_stage2.py

# 输出示例：
# ============================================================================
# 第二阶段筛选结果
# ============================================================================
# 筛选条件: Mark Minervini 8条标准
# 筛选结果: 共 1398 只股票
# ============================================================================
# 
# Top 30 股票:
# 
# 代码       名称            收盘价     MA50     位置%     涨幅%        成交量
# --------------------------------------------------------------
# 600028   中国石化         7.82     6.22  100.0%   55.0%   150244万
# 601872   招商轮船        19.65    11.07  100.0%  253.7%    37955万
# ...
```

## 示例 2：严格筛选

```bash
# 使用更严格的条件筛选
# --min-low-ratio 1.3: 股价必须比52周低点高出30%以上
# --min-high-ratio 0.80: 股价必须在52周高点的80%以内

python3 ~/screen_stage2.py --min-low-ratio 1.3 --min-high-ratio 0.80
```

## 示例 3：保存结果到指定文件

```bash
# 保存结果到指定CSV文件
python3 ~/screen_stage2.py --output ~/my_stage2_results.csv
```

## 示例 4：查看筛选结果

```bash
# 查看CSV结果文件
cat ~/.vntrader/screen_results/stage2_*.csv | head -20

# 用Excel打开
open ~/.vntrader/screen_results/stage2_*.csv
```

## 示例 5：结合其他策略

```bash
# 执行多个策略筛选
python3 ~/screen_stage2.py
python3 ~/screen_near_year_high.py

# 找出同时符合两个策略的股票
# 可以在CSV结果中进行交叉对比
```

## 筛选结果字段说明

| 字段 | 说明 |
|------|------|
| symbol | 股票代码 |
| name | 股票名称 |
| exchange | 交易所 |
| close | 收盘价 |
| ma50 | 50日均线 |
| ma150 | 150日均线 |
| ma200 | 200日均线 |
| high_250 | 250日最高价 |
| low_250 | 250日最低价 |
| position_ratio | 位置比率（收盘价/250日最高价） |
| low_ratio | 涨幅比率（收盘价/250日最低价） |
| trade_date | 交易日期 |
| volume | 成交量 |
