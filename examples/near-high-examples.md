# 接近一年新高策略使用示例

## 示例 1：基础筛选

```bash
# 执行接近一年新高筛选（默认90%阈值）
python3 ~/screen_near_year_high.py

# 输出示例：
# ====================================================================================
# 接近一年新高筛选结果
# ====================================================================================
# 筛选条件: 收盘价 >= 250日最高价 × 90%
# 筛选结果: 共 845 只股票
# ====================================================================================
# 
# Top 20 股票:
# 
# 代码       名称            收盘价    250日高     位置%   新高 日期
# -------------------------------------------------------------
# 601083   锦江航运        13.73    13.73  100.0%    ★ 2026-03-03
# 603619   中曼石油        43.99    43.99  100.0%    ★ 2026-03-03
# ...
```

## 示例 2：严格筛选

```bash
# 使用更严格的阈值筛选（95%）
python3 ~/screen_near_year_high.py --threshold 0.95

# 输出示例：
# 筛选结果: 共 432 只股票（位置 >= 95%）
```

## 示例 3：保存结果到指定文件

```bash
# 保存结果到指定CSV文件
python3 ~/screen_near_year_high.py --output ~/my_near_high_results.csv
```

## 示例 4：生成K线图

```bash
# 为筛选结果生成K线图
python3 ~/generate_near_high_charts.py

# 打开索引页面查看K线图
open ~/.vntrader/charts/near_year_high/index.html
```

## 示例 5：陶博士推荐用法

根据陶博士2006的建议，应该在以下板块中筛选：
- 基金持股3%以上
- 北向持股三千万以上

这样可以过滤掉流动性差的股票和基本面有问题的股票。

## 筛选结果字段说明

| 字段 | 说明 |
|------|------|
| symbol | 股票代码 |
| name | 股票名称 |
| exchange | 交易所 |
| close | 收盘价 |
| high_250 | 250日最高价 |
| position_ratio | 位置比率（收盘价/250日最高价） |
| is_new_high | 是否创一年新高（True/False） |
| trade_date | 交易日期 |
| volume | 成交量 |
| turnover | 成交额 |

## 与其他指标组合使用

### 组合1：接近新高 + 高RPS

找出接近一年新高且RPS较高的股票：

```bash
# 执行接近一年新高筛选
python3 ~/screen_near_year_high.py --threshold 0.95

# 然后检查RPS值，重点关注RPS > 80的股票
```

### 组合2：接近新高 + 成交量放大

找出接近一年新高且成交量放大的股票，这通常意味着更强的突破信号。

### 组合3：接近新高 + 净利润断层

陶博士特别推荐关注净利润断层的股票，这类股票突破后往往有更好的表现。

## 注意事项

1. **数据新鲜度**：确保数据库中的数据是最新的
2. **假突破风险**：不是所有接近新高的股票都会成功突破
3. **止损设置**：建议设置-5%到-8%的止损位
4. **仓位控制**：单只股票仓位不超过10%
