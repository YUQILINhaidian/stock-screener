## 示例 4：陶博士推荐用法

根据陶博士2006的建议，月线反转应该结合以下条件使用：
- 基金持股3%以上
- 北向持股三千万以上
- RPS强度（RPS50>87 或 RPS120>90）

这样可以过滤掉流动性差的股票和基本面有问题的股票。

## 月线反转7条标准

1. **RPS强度**：RPS50>87 OR RPS120>90
2. **结构紧凑**：50日低点>200日低点 或 30日低点>120日低点
3. **创新高**：10天内曾创80日新高 或 当天创50日新高
4. **均线支撑**：收盘价站上20日线和200日线
5. **均线靠近**：45天内至少有一次低于200日线
6. **均线趋势**：120日线或200日线呈上升趋势
7. **价格位置**：接近120日或250日新高

## 与其他指标组合使用

### 组合1：月线反转 + 口袋支点

月线反转后出现口袋支点，是加仓的好时机。

```bash
# 执行月线反转筛选
python3 ~/screen_monthly_reversal.py

# 执行口袋支点筛选
python3 ~/screen_pocket_pivot.py

# 找出同时出现在两个筛选结果中的股票
```

### 组合2：月线反转 + 第二阶段

月线反转通常是股票进入第二阶段的早期信号。

### 组合3：月线反转 + 接近新高

月线反转且接近一年新高的股票更强。

## 买卖建议

### 买入时机
1. 出现月线反转信号当天或次日买入
2. 结合成交量放大确认
3. 可以分批建仓

### 卖出时机
1. 跌破信号最低价止损
2. 趋势破坏后减仓
3. 达到目标价后止盈

## 注意事项

1. **RPS必须设置**：必须正确计算RPS才能使用
2. **结构紧凑是关键**：避免选入结构松散的股票
3. **均线修复**：关注长期均线的修复情况
4. **市场环境**：弱势市场中信号可能失效
5. **止损严格**：一旦信号失败必须止损（-8%到-10%）
