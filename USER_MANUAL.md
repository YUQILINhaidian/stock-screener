## 🚀 快速开始

### 方式 1: 一键启动（推荐）

```bash
cd ~/.agents/skills/stock-screener
./start.sh
```

系统会自动：
- ✅ 检查依赖
- ✅ 启动后端 API（端口 8000）
- ✅ 启动前端 UI（端口 5173）
- ✅ 打开浏览器

### 方式 2: 手动启动

**启动后端**：
```bash
cd ~/.agents/skills/stock-screener/api
python3 -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

**启动前端**：
```bash
cd ~/.agents/skills/stock-screener/ui
npm run dev
```

### 方式 3: Docker 启动

```bash
cd ~/.agents/skills/stock-screener
./start.sh --docker
```

**三大核心功能**：

#### 市场环境分析
- 判断当前市场趋势（牛市/熊市/震荡市）
- 给出置信度评分
- 推荐适合的策略

- 输入股票代码
- AI 深度分析技术面
- 综合评分（0-100分）

- 支持：行业、RPS、量比、市值、突破等

**功能特性**：
- 策略历史表现验证

  - 夏普比率
  - 最大回撤
  - 胜率
  - 交易次数
  - 盈亏比
  - 卡尔玛比率

**核心功能**：

- 收益统计

## 📖 详细使用指南

### 选股中心使用流程

#### 步骤 1: 选择策略
1. 打开 http://localhost:5173/
2. 浏览 9 个策略卡片
3. 查看每个策略的：
   - 胜率
   - 平均收益
   - 适用市场
   - 策略描述

#### 步骤 2: 执行选股
1. 点击策略卡片上的 **"🚀 开始选股"** 按钮
2. 实时进度条显示选股进度
3. 等待选股完成（通常 3-5 秒）

#### 步骤 3: 查看结果
选股结果表格包含：
- **股票代码**
- **股票名称**
- **收盘价**
- **涨跌幅**
- **RPS 指标**（分级着色）
  - 🔴 RPS > 95：红色（最强）
  - 🟠 RPS > 90：橙色（强势）
  - 🟡 RPS > 80：黄色（中等）
  - ⚪ 其他：灰色
- **成交量**
- **量比**

#### 步骤 4: 后续操作
- 点击股票进入 **AI 助手** 深度分析
- 添加到 **股票池** 进行跟踪
- 在 **回测中心** 验证策略表现

### AI 助手使用指南

#### 市场环境分析

**使用场景**：
- 不确定当前市场环境
- 想知道适合哪种策略
- 需要交易决策参考

**使用步骤**：
1. 打开 http://localhost:5173/ai
2. 点击 **"🔍 分析当前市场"** 按钮
3. AI 将在 10-30 秒内返回分析结果：
   - **市场趋势**：🐂 牛市 / 🐻 熊市 / 📊 震荡市
   - **置信度**：0-100%
   - **分析理由**：详细说明
   - **推荐策略**：2-3 个策略

**示例结果**：
```
市场趋势: 📊 震荡市
置信度: 85%
分析理由: 上证指数近期在 2900-3100 点震荡，成交量萎缩，缺乏明确方向
推荐策略: 月线反转策略、蓝色钻石策略
```

#### 个股 AI 解读

**使用场景**：
- 想深入了解某只股票
- 需要买卖决策建议
- 进行技术面分析

**使用步骤**：
1. 在 **AI 助手** 页面找到"个股 AI 解读"
2. 输入股票代码（例如：`300185`）
3. 点击 **"🤖 AI 深度解读"** 按钮
4. 查看分析结果：
   - **综合评分**：0-100 分
   - **建议操作**：📈 买入 / ⏸️ 持有 / 📉 卖出
   - **技术面评价**：简短总结
   - **风险提示**：止损位、建议仓位

**示例结果**：
```
股票: 通裕重工 (300185)
综合评分: 85 分
建议操作: 📈 买入
技术面评价: 技术面强势，RPS 高位，量价配合良好
风险提示: 止损 -8%，建议仓位 10%
```

#### 自然语言选股

**使用场景**：
- 有明确的选股想法，但不想手动筛选
- 想快速验证一个选股思路

**使用步骤**：
1. 在"自然语言智能选股"区域
2. 输入您的需求，例如：
   - "找出最近突破的新能源龙头股"
   - "筛选 RPS120 大于 90，量比大于 1.5 的股票"
   - "半导体行业中市值超过 100 亿的股票"
3. 点击 **"🧠 AI 智能解析"**
4. AI 将解析并生成筛选条件

**支持的条件**：
- **行业**：新能源、半导体、医药、消费等
- **RPS**：RPS50、RPS120、RPS250 的阈值
- **量比**：成交量比值
- **市值**：large（大盘）、mid（中盘）、small（小盘）
- **涨跌幅**：今日涨跌幅
- **突破**：recent_high（近期新高）、year_high（年度新高）

### 回测中心使用指南

#### 配置回测参数

1. **选择策略**
   - 从下拉菜单选择要回测的策略

2. **设置回测周期**
   - **开始日期**：回测开始时间
   - **结束日期**：回测结束时间
   - 建议周期：至少 6 个月

3. **配置初始资金**
   - 默认：100 万元
   - 可根据实际情况调整

#### 执行回测

1. 点击 **"🚀 开始回测"** 按钮
2. 等待回测完成（通常 5-15 秒）
3. 查看回测结果

#### 解读回测指标

| 指标 | 说明 | 优秀标准 |
|------|------|---------|
| **总收益率** | 整个周期的收益 | > 20% |
| **年化收益** | 年化后的收益率 | > 15% |
| **夏普比率** | 风险调整后收益 | > 1.5 |
| **最大回撤** | 最大跌幅 | < -20% |
| **胜率** | 盈利交易占比 | > 60% |
| **交易次数** | 总交易次数 | 适中 |
| **盈亏比** | 平均盈利/平均亏损 | > 2.0 |
| **卡尔玛比率** | 收益/最大回撤 | > 1.0 |

## ❓ 常见问题

### Q1: 启动失败怎么办？

**A**: 检查以下几点：
1. 端口 8000 和 5173 是否被占用
   ```bash
   lsof -i:8000
   lsof -i:5173
   ```
2. Python 和 Node.js 是否正确安装
   ```bash
   python3 --version  # 需要 3.9+
   node --version      # 需要 18+
   ```
3. 依赖是否正确安装
   ```bash
   cd api && pip install -r requirements.txt
   cd ui && npm install
   ```

### Q2: AI 功能无法使用？

**A**: 检查：
1. 是否配置了 `LLM_API_KEY`
2. API Key 是否正确
3. 网络连接是否正常
4. 运行测试脚本：
   ```bash
   cd api && python3 test_llm.py
   ```

### Q3: 选股没有结果？

**A**: 可能的原因：
1. 没有历史数据文件
2. 数据库路径配置错误
3. 当前市场不满足策略条件

### Q4: 回测失败？

**A**: 检查：
1. 回测周期是否合理（至少 3 个月）
2. 是否有对应的历史数据
3. 数据库是否正常连接

### Q5: 如何停止服务？

**A**: 
```bash
# 停止开发模式
./start.sh stop

# 停止 Docker 模式
./start.sh stop --docker
```

### Q6: 如何查看日志？

**A**:
```bash
# 开发模式
tail -f /tmp/stock-screener-api.log
tail -f /tmp/stock-screener-ui.log

# Docker 模式
docker logs stock-screener-api
docker logs stock-screener-ui
```

### Q7: 如何更新代码？

**A**:
```bash
cd ~/.agents/skills/stock-screener
git pull  # 如果使用 Git 管理
./start.sh stop
./start.sh
```

### Q8: 支持哪些浏览器？

**A**: 推荐使用：

- ⚠️ IE 不支持

### Q9: 能在移动设备上使用吗？

**A**: 可以！前端采用响应式设计，完美适配：
- 📱 手机

## 💡 最佳实践

### 选股流程建议

1. **早盘前**（9:00 前）
   - 使用 AI 助手分析市场环境
   - 根据推荐选择合适的策略

2. **开盘后**（9:30-10:00）
   - 执行选股
   - 查看选股结果
   - 对重点股票进行 AI 解读

3. **盘中监控**（10:00-14:30）
   - 跟踪股票池中的持仓
   - 根据涨跌调整仓位

4. **收盘后**（15:00 后）
   - 更新股票池数据
   - 回顾当日表现
   - 准备第二天策略

### 风险控制建议

1. **仓位管理**
   - 单只股票不超过 10%
   - 同一板块不超过 30%
   - 保留 20% 现金仓位

2. **止损止盈**
   - 设置 -8% 止损线
   - 设置 +20% 止盈线
   - 严格执行纪律

3. **分散投资**
   - 持有 5-10 只股票
   - 不同行业分散
   - 不同策略组合
