## 🌟 陶博士三大核心买点

陶博士的选股体系中，有三个核心买点，被认为是最有效的买入时机：

### 1️⃣ 月线反转（Monthly Reversal）

**📖 详细文档**：[monthly-reversal-guide.md](./references/monthly-reversal-guide.md)

```
📈 核心逻辑：捕捉股票从底部区域开始反转上升的早期信号

✅ 7大筛选条件：
1. RPS强度：RPS50>87 或 RPS120>90
2. 结构紧凑：底部不断抬高
3. 创新高：10天内创80日新高
4. 均线支撑：站上20日和200日线
5. 均线靠近：刚从均线下方站上来
6. 均线趋势：长期均线开始上升
7. 价格位置：接近近期高点

🎯 最佳买点：信号出现当天或次日
🛑 止损位：-8%到-10%
💰 目标收益：50%-100%+
⏱️ 持仓周期：3-6个月

📊 实盘数据（2026-04-08）：
- 平均收益：+10.67%
- 胜率：70% (23/33只)
- 最佳：+117.15%
- 最差：-20.23%
```

**典型形态**：
```
价格走势：📉📉📉 → 🎯月线反转 → 📈📈📈
均线状态：空头排列 → 均线靠拢 → 多头排列
```

**执行命令**：
```bash
python3 ~/screen_monthly_reversal.py
```

### 2️⃣ 口袋支点（Pocket Pivot）

**📖 详细文档**：[pocket-pivot-guide.md](./references/pocket-pivot-guide.md)

```
📈 核心逻辑：捕捉主力资金吸筹信号，成交量突破

✅ 9大筛选条件：
1. RPS强度：RPS250≥87 或 RPS120≥90 或 RPS50≥90
2. 成交量放大：创10日最高成交金额
3. 均线突破：创90/100/120日新高
4. 结构紧凑：15日内最低价不是50日内最低价
5. 调整幅度限制：阶段最大下跌不超过46%
6. 结构紧凑条件：满足特定价格结构
7. 涨幅要求：当日涨幅≥5%
8. 换手率稳定：两天平均换手≤15%
9. 价格偏离限制：控制回撤幅度

🎯 最佳买点：成交量异常放大当天
🛑 止损位：-5%到-8%
💰 目标收益：30%-60%
⏱️ 持仓周期：1-3个月

📊 实盘数据（2026-04-08）：
- 平均收益：-11.16%
- 胜率：19% (22/114只)
- 最佳：+42.83%
- 最差：-35.40%
```

**典型形态**：
```
价格走势：调整震荡 → 🎯成交量爆发 → 突破上涨
成交量：  萎缩      → 异常放大     → 持续放大
```

**执行命令**：
```bash
python3 ~/screen_pocket_pivot.py
```

**⚠️ 注意**：口袋支点策略近期表现不佳，建议与月线反转组合使用。

## 📚 经典趋势跟踪策略

### 🔹 第二阶段策略（Stage 2）

**📖 详细文档**：[stage2-guide.md](./references/stage2-guide.md)

```
📈 核心逻辑：《股票魔法师》Mark Minervini 四阶段模型

✅ 8大标准：
1. 股价 > 150日和200日均线
2. 150日均线 > 200日均线
3. 200日均线上涨至少1个月
4. 50日均线 > 150日和200日均线
5. 股价比52周低点高出至少25%
6. 股价处在52周高点的25%以内
7. RS排名不低于70（最好90左右）
8. 股价在50日均线之上

🎯 最佳买点：进入第二阶段初期
🛑 止损位：-8%到-10%
💰 目标收益：50%-150%
⏱️ 持仓周期：3-6个月
```

**四阶段模型**：
```
第一阶段：忽略时期（底部巩固）
第二阶段：突围时期（加速上涨）← 最佳买入
第三阶段：到顶时期（分配利润）
第四阶段：衰败时期（投降下跌）
```

**执行命令**：
```bash

## 🎓 策略选择指南

### 根据投资风格选择

| 投资风格 | 推荐策略 | 理由 |
|----------|----------|------|
| **价值+趋势** | 月线反转 | 底部反转，安全边际高 |
| **趋势跟踪** | 第二阶段、接近新高 | 顺势而为，趋势明确 |
| **波段交易** | 火车头、蓝色钻石 | 中短期机会，灵活操作 |
| **激进短线** | 首次涨停 | 快进快出，短线爆发 |
| **稳健长线** | 月线反转 + 第二阶段 | 双重确认，成功率高 |

python3 ~/generate_monthly_reversal_charts.py

# 5. 查看持仓收益

```
